# ¿Cómo ingresar al sistema de facturación?

Para ingresar al Sistema se debe ingresar a **Windows(Win)**; escribimos **“Kodigo Fuente”**, y allí aparecerá una lista donde podrás seleccionarlo usando las **teclas de dirección o el mouse** (hasta que quede sombreado) y presionas **Enter** o simplemente debes darle clic.

![busca kodigo fuente entre tus aplicaciones](/img/inicio_sesion/busqueda_inicial.png)

Al ingresar a la aplicación podrá salir un mensaje de permiso de usuario debes darle clic en el botón si, para poder continuar. Para ingresar al sistema debe ingresar el usuario y contraseña de acceso, previamente asignados.

:::info Nota:
Presionando la tecla **(TAB ↹)** puedes pasar del campo Usuario al campo Contraseña y una vez ingrese los datos correctamente, presiona **ENTER (INTRO)** o dar clic sobre el botón **"Ingresar"**.
:::

![inicio de sesión](/img/inicio_sesion/iniciar_sesion.png)

Una vez el ingreso sea exitoso, se mostrara el menú principal, que aparece en el recuadro Azul del costado izquierdo, con los siguientes módulos:

1.  Facturación
2.  Terceros
3.  Reportes
4.  Administración
5.  Herramientas
6.  Sistema
7.  Salir

Para desplazarse entre cada módulo debe presionar la tecla que corresponda a la letra inicial de cada palabra. Por ejemplo si desea moverse al módulo **Administración**, presione la tecla **A**; si desea moverse al módulo **Facturación**, presiona la tecla **F** y así sucesivamente.

![menu_principal](/img/inicio_sesion/menu_principal.png)
